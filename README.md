
# csv-make

Generate a [CSV][] file of [user accounts for upload][upload] to [Moodle][].

## Install .. build

```
npm install

npm run copy-config

vi ./config.json

npm run build

npm test  # Run JSON lint.
```

You may find this [JSON validator][] useful! :heart:
